# نقش الاسم — مولّد الخط العربي

أداة ويب بصفحة واحدة لتوليد نقش بأسماء عربية بخطوط أصيلة، مع معاينة حية وتشكيل الحروف حرفًا حرفًا، وتصدير الصورة بمقاسات وسائل التواصل.

**Naqsh al-Ism** — a single-page web app that turns an Arabic name into calligraphic artwork: live preview, letter-by-letter "writing" animation, and one-click PNG export at social-media sizes.

## المميزات / Features

- ✍️ معاينة حية للاسم بأكثر من ١١ خطًّا عربيًّا (أميري، كوفي، الرقعة، نستعليق…)
- 🎨 خلفيات متعددة بطابع المخطوطات (مخطوطة، رق أثري، سلطانية، زمردية…)
- 🖼️ ثلاثة مقاسات للتصدير: بروفايل ١٠٨٠×١٠٨٠، قصة/ريلز ١٠٨٠×١٩٢٠، تغريدة ١٢٠٠×٦٣٠
- 🪶 تشكيل الحروف حرفًا حرفًا بحركة قلم، مع تحكّم في السرعة
- ⬇️ تحميل الناتج كصورة PNG عالية الدقة

## التشغيل / Run

ملف ثابت واحد — افتح `index.html` في المتصفح مباشرة، أو قدّمه عبر أي خادم ثابت:

```bash
npx serve .
```

## البنية / Stack

HTML + CSS + Canvas (vanilla JavaScript) — بدون أي اعتماديات بناء. الخطوط من Google Fonts.
